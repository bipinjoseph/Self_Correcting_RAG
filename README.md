Self-Correcting RAG System

A Retrieval-Augmented Generation (RAG) project built using Python, Gemini API, LangChain, FAISS, Streamlit, and a self-verification loop to reduce hallucination and improve factual accuracy.

Features

Document ingestion and chunking

Embedding generation using Gemini

Vector search using FAISS

RAG pipeline with context injection

Self-correcting loop (answer → verify → correct)

Streamlit interface for easy interaction

Tech Stack

Google Gemini API

LangChain

FAISS (vector database)
